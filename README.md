## Instalação

## FRONTEND

1. `$ cd frontEnd`
2. `$ npm install`
3. `$ npm run dev`

## BackEnd

Database data `server/app.js`
`$ cd server`
`$ npm install`
`$ npm start`

## BackEnd

- Cria Cliente no banco de dados
- Ler Cliente no banco de dados
- Atualizar Cliente no banco de dados
- Excluir Cliente no banco de dados
- Clique no cartão para editar e excluir
